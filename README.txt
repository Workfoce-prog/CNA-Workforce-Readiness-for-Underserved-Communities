
# CNA Student Progress Dashboard

This Streamlit app provides an interactive dashboard for monitoring student progress across all four sections of the CNA curriculum:
1. Foundational Skills
2. Core Training Modules
3. Social & Emotional Support
4. Career & Life Skills

## Files Included
- `cna_dashboard_app.py`: Streamlit app source code
- `CNA_Curriculum_Dashboard_Data.csv`: Full data with RAG scores and recommendations
- `README.txt`: This file

## How to Run the App

1. Install Streamlit:
   ```
   pip install streamlit
   ```

2. Run the app:
   ```
   streamlit run cna_dashboard_app.py
   ```

3. Use the sidebar filters to explore student performance by section or RAG status.
