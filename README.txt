# CNA Student Learning App

An interactive, student-friendly portal for CNA learners to:

- Explore curriculum topics
- Take quizzes and get instant feedback
- Track their progress with RAG status
- Receive actionable recommendations and motivation

## To Run Locally

1. Install Streamlit:
```
pip install -r requirements.txt
```

2. Run the app:
```
streamlit run cna_student_learning_app.py
```

## To Deploy on Streamlit Cloud

1. Push this folder to a public GitHub repository.
2. Go to [https://streamlit.io/cloud](https://streamlit.io/cloud)
3. Choose this repo, select `cna_student_learning_app.py` as the main file, and deploy.
